# Fraud Detection 💳

## Overview
Built a Credit Card Fraud Detection model achieving 93% accuracy using Machine Learning.

## Results
| Model | Accuracy |
|---|---|
| Random Forest | 92% |
| Gradient Boosting | 93% 🏆 |

## Key Highlights
- Handled highly imbalanced dataset (0.17% fraud)
- Used undersampling to balance 492 fraud vs 492 normal
- 96.56% confidence on fraud detection
- Feature importance analysis

- Confusion Matrix Visualization
